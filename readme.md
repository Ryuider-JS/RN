📦 다음 구매일 예측 (purchase_history.xlsx 기반)
이 코드는 과거 구매 이력을 분석하여 Q코드별 다음 예상 구매일을 예측하는 파이썬 스크립트입니다.
입력 파일은 purchase_history.xlsx이며, 결과는 다음_구매일_예측결과.xlsx로 저장됩니다.

---

🔧 주요 단계
1️⃣ 라이브러리 설치 및 불러오기
pandas, numpy, scikit-learn, openpyxl 사용
데이터 분석과 엑셀 파일 처리를 위한 기본 패키지 설치

2️⃣ 데이터 불러오기 및 전처리
엑셀 파일(purchase_history.xlsx)을 불러옴
주문 일시 컬럼을 datetime 형식으로 변환
Q코드, 주문 일시 기준으로 정렬

3️⃣ 구매 주기 분석
Q코드별로 데이터를 그룹화
각 그룹에서 주문 간격(일)을 계산
최소 2회 이상의 구매 이력이 있는 경우만 예측 수행

4️⃣ 예측 모델 적용
LinearRegression을 이용해 구매 주기 변화 패턴 학습
최근 주문일에 예측된 간격을 더해 다음 구매일 계산

5️⃣ 결과 저장
결과 컬럼:
Q코드
최근 주문일
평균 구매 간격(일)
예상 다음 구매일
결과를 다음_구매일_예측결과.xlsx로 저장

---

🧮 출력 예시
| Q코드 | 최근 주문일 | 평균 구매 간격(일) | 예상 다음 구매일 |
|:------|:------------|:------------------:|:------------------|
| Q12345 | 2024-12-01 | 32.5 | 2025-01-02 |
| Q67890 | 2025-02-14 | 27.0 | 2025-03-13 |
